PDA Reference

Formal definition. A PDA is a 7-tuple \( M = (Q, \Sigma, \Gamma, \delta, q_0, Z_0, F) \) (standard notation).

\(Q\) — finite set of states
\(\Sigma\) — input alphabet
\(\Gamma\) — stack alphabet
\(\delta\) — transition function
\(q_0\) — start state
\(Z_0\) — initial stack symbol
\(F\) — accept states
